#### 安装node和npm 
- 先下载node
[node下载](https://nodejs.org/zh-cn/)
- 下载完之后将node解压的desktop,然后将文件夹更改为node
- 之后添加到启动
```
sudo ln -s /home/jp/Desktop/node/bin/node /usr/local/bin/node
sudo ln -s /home/jp/Desktop/node/bin/npm /usr/local/bin/npm
```
